# osc-send

```
Send an OSC message to a remote address

Usage: osc-send [OPTIONS] --address <ADDRESS> <COMMAND> <VALUE>

Commands:
  int     
  float   
  string  
  bool    1 for true, 0 for false
  help    Print this message or the help of the given subcommand(s)

Options:
  -r, --remote <REMOTE>    [default: 127.0.0.1:9000]
  -a, --address <ADDRESS>  
  -h, --help               Print help
  -V, --version            Print version
Send an OSC message to a remote address
```
